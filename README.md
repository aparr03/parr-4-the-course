# Parr-4-The-Course
A simple cookbook website

HOW TO SET UP AND EDIT CODE
1. Install Node.js
   a. https://nodejs.org/en
   b. Verify node in terminal: "node -v"
   c. Verify npm in terminal: "npm -v"
   d. Verify npx in termianl: "npx --version"
   e. If npx hitches, run "npx claer-npx-cache"

2. Install VS Codepp
   a. https://code.visualstudio.com/download

4. Github Desktop
   a. Download Github Desktop: https://desktop.github.com/download/
   b. Clone the repo parr-4-the-course and save it to your machine
   c. Pull the latest edition of main
   d. Open the file directory in VS Code

5. Vercel Config
   a. Navigate to Vercel and link your Github account: https://vercel.com/
   b. Join my project workspace via invite
   c. In terminal, run "npm install -g vercel"

6. Previewing the site
   *There are a few ways you can do this*
   a. Run "npm start" in terminal (this will host the website locally from your machine using Node)
   b. Run "vercel dev" in terminal (This will use the Vercel CLI and prompt you int nermal to use y/n and arrow keys to select your project)

7. Pushing Changes
   a. Save your code in VS Code
   b. Open Github Desktop to see the changes you made in each file
   c. Add a Comment and Description in the lower left-hand corner to explain what you changed
   d. Click "Commit"
   e. In the top, it should now say "Push Origin" rather than "Pull" or "Fetch". Click "Push Origin"

Congratulations, you can now successfully set up, edit, preview, and push your edits to the website!!!
